## 🎯 규칙 적용 범위

### 전역 규칙
- 코딩 스타일 및 포맷팅
- Git 워크플로우
- 문서화 표준
- 보안 가이드라인

### 기술별 규칙
- **Python**: 가상환경, 패키지 관리, 테스팅
- **JavaScript/TypeScript**: 린팅, 번들링, 프레임워크
- **Git**: 브랜치 전략, 커밋 메시지 규칙

### AI 도구 규칙
- GitHub Copilot 최적화
- MCP 서버 연결 및 설정
- 자동화 워크플로우

## 🚀 사용 방법

### 규칙 적용
1. 새 프로젝트 시작 시 해당 규칙 문서 참조
2. 코드 리뷰 시 규칙 준수 확인
3. CI/CD 파이프라인에 규칙 검사 통합

### 규칙 업데이트
```powershell
# 최신 규칙 가져오기
git pull origin main

# 규칙 변경사항 확인
git diff HEAD~1 rules/
```
